# Contributing to Popsy.js 🎉

Thanks for your interest in contributing! Everyone is welcome to add new animations, fix bugs, or improve existing components.

---

## How to Contribute

1. **Fork** this repository
2. Make your changes
3. **Open a Pull Request** with a short description of what you added

---

## Adding a New Component

1. Create your component file inside the `components/` folder — e.g. `components/typewriter.js`
2. Write full docs inside the file as comments (look at `confetti.js` or `toast.js` as reference)
3. Add your function to `popsy.js` as well
4. Update the components table in `README.md`

---

## Rules

- Keep it simple — Popsy.js is beginner friendly
- Test your component before submitting

---

## Ideas for New Components

- Typewriter effect
- Smooth scroll
- Custom cursor
- Shake animation
- Fade in / slide up
- Tooltip
- Confetti shapes (stars, hearts)

Feel free to pick one or come up with your own!
