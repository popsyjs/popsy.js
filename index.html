Popsy.js

A lightweight JavaScript library for fun web animations. One line, zero setup.

<script src="https://cdn.jsdelivr.net/gh/popsyjs/popsy.js@main/popsy.js"></script>
Components
🎉 Confetti
Burst of colorful dots from the center of the screen. Click a button to preview and see its code.
Popsy.confetti()
Popsy.confetti({ count: 200 })
Popsy.confetti({ colors: ['#f7c948', '#ff6b6b', '#fff'] })
🔔 Toast
Notification popup in the bottom-right corner. Click a button to preview and see its code.
Popsy.toast('Saved successfully!')
Popsy.toast('Something went wrong', { type: 'error' })
Popsy.toast('Did you know?', { type: 'info' })
Popsy.toast('Stays for 6 seconds!', { duration: 6000 })
📋 Copy
Creates a styled copy button and injects it anywhere. Three presets: default, api, simple — all fully customizable.
preset: 'default'
npm install popsy.js
preset: 'api'
sk-••••••••••••xK9f
preset: 'simple'
https://mysite.com/ref/abc123
custom — bg, color, radius
Custom styled button
Popsy.copy({ value: 'npm install popsy.js', target: '#myBox' })
Popsy.copy({ value: 'sk-abc123xyz', target: '#keyBox', preset: 'api', label: 'Copy API key', copied: 'Copied!' })
Popsy.copy({ value: 'https://mysite.com/ref/abc123', target: '#linkBox', preset: 'simple' })
Popsy.copy({ value: 'Your custom value', target: '#customBox', bg: '#6366f1', color: '#ffffff', radius: '20px', label: 'Copy invite link', copied: 'Done!' })
💬 Tooltip
Lightweight tooltip that attaches to any element. Hover the buttons below to preview each position.
Popsy.tooltip('#myBtn', 'Save your changes')
Popsy.tooltip('#myBtn', 'Click to expand', { position: 'bottom' })
Popsy.tooltip('#myBtn', 'Go back', { position: 'left' })
Popsy.tooltip('#myBtn', 'Open settings', { position: 'right' })
Popsy.tooltip('#myBtn', 'Pro feature', { bg: '#6366f1', color: '#fff' })
Popsy.tooltip('#myBtn', 'Your profile', { delay: 300 })
